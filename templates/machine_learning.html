Tutorial: Machine Learning

Come utilizzare la pagina di Machine Learning

Questa pagina ti permette di caricare un dataset, analizzarlo e creare modelli di machine learning per classificazione o regressione in pochi semplici passaggi:

  1. Carica il dataset

    Carica un file CSV contenente i tuoi dati. Il sistema supporta dataset con valori numerici e categoriali.

  2. Analizza il dataset

    Dopo aver caricato il file, clicca su "Analizza Dataset". Il sistema utilizzerà OpenAI per generare una descrizione dettagliata del tuo dataset.

  3. Seleziona la colonna target

    Scegli quale colonna vuoi prevedere dal menu a tendina e clicca su "Analizza". Il sistema determinerà automaticamente se si tratta di un problema di classificazione o regressione.

  4. Raccomandazione algoritmo

    Il sistema ti suggerirà l'algoritmo più adatto in base al tipo di dati e al problema da risolvere.

  5. Training del modello

    Clicca sul pulsante "TRAIN" per addestrare il modello sui tuoi dati. Il sistema dividerà automaticamente i dati in set di training e test.

  6. Visualizzazione e predizione

    Dopo il training, potrai visualizzare i risultati tramite grafici di dispersione e curve di fit. Inoltre, potrai inserire nuovi dati per ottenere predizioni dal modello addestrato.

Tipi di problemi supportati

  • Classificazione: Quando la colonna target contiene valori categoriali (es. "sì"/"no", "rosso"/"verde"/"blu").
  • Regressione: Quando la colonna target contiene valori numerici continui (es. prezzo, temperatura).

Suggerimento

Per ottenere risultati migliori, assicurati che il tuo dataset sia pulito e non contenga valori mancanti. Se necessario, il sistema tenterà di gestire automaticamente i valori mancanti e le conversioni di tipo.
