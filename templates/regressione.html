Guida all'utilizzo dello strumento di Regressione Lineare

1. Cos'è la Regressione Lineare?

La regressione lineare è una tecnica statistica che permette di modellare la relazione tra una variabile dipendente (y) e una o più variabili indipendenti (x). Il modello risultante è una linea che meglio si adatta ai dati.

  • Utile per prevedere valori futuri basandosi su dati esistenti
  • Ideale per identificare relazioni lineari tra variabili
  • Fornisce informazioni sulla forza della relazione tra le variabili

2. Preparazione dei Dati

Per utilizzare questo strumento, è necessario fornire un dataset in formato CSV contenente almeno due colonne numeriche:

  • Una colonna per la variabile indipendente (X)
  • Una colonna per la variabile dipendente (Y)
  • I dati devono essere puliti (senza valori mancanti o anomali)
  • È consigliabile avere intestazioni di colonna

3. Utilizzo dello Strumento

  1. Scegli se utilizzare un dataset personalizzato o di esempio
  2. Se personalizzato, carica il tuo file CSV
  3. Seleziona quali colonne utilizzare come X e Y
  4. Fai clic su "Addestra Modello" per generare la regressione
  5. Visualizza i risultati nel grafico e nella tabella dei parametri
  6. Utilizza il campo di input per effettuare nuove previsioni

4. Interpretazione dei Risultati

  • Coefficiente Angolare (m): Indica quanto varia Y al variare di X. Un valore positivo mostra una relazione crescente, negativo una decrescente.
  • Intercetta (q): Il valore di Y quando X è zero.
  • Coefficiente di Determinazione (R²): Misura quanto bene il modello spiega i dati (da 0 a 1). Più è vicino a 1, migliore è l'adattamento.
  • Errore Quadratico Medio (MSE): L'errore medio del modello. Valori più bassi indicano previsioni più precise.
